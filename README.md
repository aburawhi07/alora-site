# Aloragraphic

A modern, interactive web application for graphic design and visualization built with React and Vite.

## 🚀 Live Demo

[View the live application](https://aloragraphic.vercel.app)

## 📋 About

Aloragraphic is a professional-grade graphic design tool that combines an intuitive user interface with powerful design capabilities. Whether you're creating graphics for social media, presentations, or personal projects, Aloragraphic provides the tools you need.

## ✨ Features

- Modern, responsive design interface
- Interactive canvas for graphic creation
- Real-time preview
- Fast build times with Vite
- ESLint configuration for code quality

## 🛠️ Tech Stack

- **Frontend Framework**: React 18
- **Build Tool**: Vite
- **Styling**: CSS
- **Deployment**: Vercel

## 📦 Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/aburawhi07/aloragraphic.git
cd aloragraphic
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

## 📚 Available Scripts

- `npm run dev` - Start development server with HMR
- `npm run build` - Build for production
- `npm run preview` - Preview production build locally
- `npm run lint` - Run ESLint to check code quality
